# Structures in C++

## 📌 What is a Structure?

A structure (struct) is a user-defined data type
that allows grouping variables of different data types
under a single name.

It is mainly used to represent a record.

---

## 🧱 Basic Syntax

struct StructureName {
    dataType member1;
    dataType member2;
};

---

## 🧠 Why Use Structures?

- Organize related data
- Improve code readability
- Model real-world entities

Example:
Student → name, rollNumber, marks

---

## 🔄 Accessing Members

Use the dot (.) operator:

objectName.memberName

---

## ⚠ Important Notes

- Structures can contain functions.
- Structures support constructors.
- In C++, struct and class are almost the same.
